# Ubuntu Bridge Initiative – Cybersecurity Internship

## About

This repository documents my work throughout the Ubuntu Bridge Initiative Cybersecurity Internship.

The program focuses on practical cybersecurity skills through hands-on investigations, security assessments, incident response, cloud security, detection engineering and professional reporting.

Each stage contains the reports, notes, scripts and documentation produced during the internship.

---

## Objectives

- Develop practical SOC Analyst skills
- Perform Digital Forensics and Incident Response (DFIR)
- Investigate malware and attack chains
- Practice detection engineering
- Strengthen Linux and Cloud Security knowledge
- Produce professional cybersecurity reports
